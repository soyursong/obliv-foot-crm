원내 도장 이미지 폴더
=====================

파일명: jongno-foot-stamp.png
용도: 서류 출력 시 자동 오버레이되는 원내 도장 이미지

요건:
- 형식: PNG (투명 배경 권장)
- 권장 해상도: 300×300 px 이상
- 위치(인쇄 시 기본값): 각 양식 우하단 (right:52px, bottom:52px, width:88px)
- 위치 조정: field_map stamp_x / stamp_y 좌표로 양식별 커스텀 가능 (Phase 2)

수급: 오블리브 풋센터 종로 원내 도장 스캔 or 디지털 파일 → 본 폴더에 저장
담당: 현장(김주연) 또는 운영팀 전달 후 dev-foot 적용
