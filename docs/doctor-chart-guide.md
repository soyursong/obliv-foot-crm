## 1. 진료차트 여는 방법 (경로)

진료차트는 **3곳**에서 열 수 있습니다.

### 방법 A — 대시보드(칸반)에서 바로 열기

1. 왼쪽 메뉴에서 **대시보드** 클릭 (URL: `/admin`)
2. 칸반의 **환자 이름 위에서 우클릭(또는 롱프레스)** → 팝업 메뉴에서 **"진료차트"** 선택
   - 또는 환자 카드를 클릭하여 상세 시트 열기 → **"진료차트"** 버튼(청록색 청진기 아이콘) 클릭
3. 화면 오른쪽에서 진료차트 Drawer(서랍)가 슬라이드로 열립니다

### 방법 B — 고객관리에서 열기

1. 왼쪽 메뉴 → **고객관리** (URL: `/admin/customers`)
2. 고객 이름 **우클릭** → 메뉴에서 **"진료차트"** 선택

### 방법 C — 예약관리에서 열기

1. 왼쪽 메뉴 → **예약관리** (URL: `/admin/reservations`)
2. 예약 클릭 → 상세에서 **"진료차트"** 버튼 선택

> **팁**: 진료차트는 항상 오른쪽에서 슬라이드로 열립니다. 바깥 어두운 영역을 클릭하거나 ESC 키를 누르면 닫힙니다.

## 2. 빠른처방 버튼 사용법

빠른처방 버튼은 자주 쓰는 처방을 **한 번 클릭**으로 입력하는 단축 기능입니다.

### 사용 경로

왼쪽 메뉴 → **진료 도구** (URL: `/admin/doctor-tools`)

→ **"진료 환자 목록"** 탭 클릭

### 사용 방법

1. 오늘 접수된 환자 목록이 표시됩니다
2. 처방이 필요한 환자 행의 **오른쪽 ▼ 화살표** 클릭 → 빠른처방 버튼 영역이 펼쳐집니다
3. 원하는 처방 버튼 클릭 → 즉시 처방 입력 완료

### 원장(의사) 클릭 시

→ 처방이 **바로 확정(초록 체크)** 됩니다

### 치료사 클릭 시

→ 처방이 **임시 저장(노란 테두리)** 됩니다 — 원장 확인 후 확정 필요

> **빠른처방 버튼이 안 보일 때**: 어드민에서 버튼을 아직 등록하지 않은 경우입니다.
> → `진료 도구` 화면 → **빠른처방 버튼** 탭에서 등록하세요 (admin/manager 전용).

## 3. 원장 모드 vs 치료사 모드

CRM은 **로그인 계정의 역할**을 자동으로 감지합니다. 별도 설정이 필요 없습니다.

| 역할 | 모드 | 처방 입력 결과 |
|------|------|--------------|
| admin (관리자) | 원장 모드 | 클릭 즉시 **확정** |
| manager (매니저) | 원장 모드 | 클릭 즉시 **확정** |
| director (원장) | 원장 모드 | 클릭 즉시 **확정** |
| therapist (치료사) | 치료사 모드 | 임시 저장 후 **원장 확인 대기** |
| 기타 직원 | 치료사 모드 | 임시 저장 후 **원장 확인 대기** |

### 현재 모드 확인 방법

`진료 도구` → `진료 환자 목록` 탭 상단 오른쪽에 작은 배지로 표시됩니다:

- **청록(teal) 배지** "의사 모드 — 바로 확정" → 원장 모드
- **노란(amber) 배지** "치료사 모드 — 임시 처방" → 치료사 모드

## 4. 임시 처방 — 노란 테두리의 의미

치료사가 빠른처방을 입력하면 해당 환자 행이 **노란 테두리**로 강조됩니다.

```
⚠ 임시 처방 대기 상태 → 원장 확인 필요
```

### 원장이 확인하는 방법

1. `진료 환자 목록` 탭에서 노란 테두리 환자 행의 **▼ 화살표** 클릭
2. 처방 내용 확인 후 **청록 "확정" 버튼** 클릭
3. 테두리가 초록으로 바뀌고 "처방 확정 완료"로 표시됩니다

### 필터 기능 활용

목록 상단의 **"임시 (N)"** 버튼을 클릭하면 임시 처방 대기 환자만 필터링할 수 있습니다.

## 5. 상용구 단축어 (//) 트리거 사용법

> **현재 구현된 트리거: `//` (슬래시 두 번)**
>
> 진료차트의 **임상경과** 텍스트 입력창에서 `//` 를 입력하면 상용구 목록이 팝업으로 표시됩니다.
> `//풋재` 처럼 단축어를 이어서 입력하면 해당 단축어로 시작하는 상용구만 필터됩니다.

### 사용 방법

1. 진료차트 Drawer 열기 → 중앙 폼의 **"임상경과"** 텍스트창 클릭
2. `//` 입력 → 단축어가 등록된 상용구 목록이 자동 표시
3. 단축어를 이어 입력하면 필터링 (예: `//족부` → "족부"로 시작하는 상용구만 표시)
4. 목록에서 원하는 항목 클릭 → `//단축어` 부분이 상용구 전체 문구로 자동 대체됩니다

### 단축어 입력 예시

| 입력 | 결과 |
|------|------|
| `//` | shortcut_key가 등록된 진료차트 상용구 전체 표시 |
| `//족부` | shortcut_key가 '족부'로 시작하는 상용구 필터링 |
| `//풋재` | shortcut_key가 '풋재'로 시작하는 상용구 필터링 |
| `//초진` | shortcut_key가 '초진'으로 시작하는 상용구 필터링 |

### 상용구 단축어가 작동하지 않을 때 체크 사항

1. **상용구가 등록되어 있는지** 확인 → 진료 도구 → 상용구 탭
2. **phrase_type이 "진료차트"로 설정되어 있는지** 확인 ← 가장 흔한 원인
   - "펜차트" 타입으로 등록된 상용구는 임상경과 자동완성에 나타나지 않습니다
3. **shortcut_key(단축어)가 입력되어 있는지** 확인
   - 단축어가 비어 있는 상용구는 `//` 팝오버에 표시되지 않습니다

### 우측 패널의 상용구 탭 (체크박스 다중 선택)

`//` 트리거 외에도 **우측 패널 → 상용구 탭**에서 체크박스로 여러 개를 선택하고 한 번에 삽입할 수 있습니다:

1. 진료차트 우측 패널 → **상용구 탭** (책 아이콘)
2. 원하는 상용구 체크박스 선택 (복수 선택 가능)
3. 하단 **"선택한 N개 임상경과에 삽입"** 버튼 클릭

## 6. 어드민에서 상용구·처방세트 관리 (CRUD)

> **접근 권한**: admin / manager 계정만 가능합니다.

### 진입 경로

왼쪽 메뉴 → **진료 도구** (URL: `/admin/doctor-tools`)

### 탭별 기능 안내

| 탭 이름 | 기능 |
|--------|------|
| **상용구** | 자주 쓰는 진료 문구 등록·수정·삭제. `//단축어`도 여기서 설정 |
| **처방세트** | 약품 조합 세트 등록·수정·삭제 |
| **진료세트** | 시술 조합 세트 등록 |
| **수가세트** | 결제 미니창의 수가항목 세트 등록 |
| **서류 템플릿** | 진단서·소견서 등 문서 양식 관리 |
| **빠른처방 버튼** | 한 클릭 처방 버튼 이름·아이콘·처방세트 연결 |

### 상용구 추가 예시 (`//풋재` 단축어 등록)

1. `진료 도구` → **상용구** 탭
2. 오른쪽 위 **"+ 상용구 추가"** 버튼 클릭
3. **유형 선택**: `진료차트` 선택 ← **반드시 '진료차트'를 선택해야 `//` 트리거에 표시됩니다**
4. **카테고리** 선택 (차팅 / 처방 / 원장님 / 일반)
5. **이름** 입력 (예: "족부 재진 경과 기본")
6. **내용** 입력 (진료 기록에 삽입될 전체 텍스트)
7. **단축어** 입력 (예: `풋재`) → `//풋재` 입력 시 이 상용구가 뜹니다
8. **활성화** 토글 켜기
9. **저장** 클릭

> **유형 안내**:
> - **진료차트** → 진료차트 임상경과 `//` 자동완성에 표시. 우측 상용구 패널에도 표시
> - **펜차트** → 펜차트 도구에서만 사용. 진료차트에는 나타나지 않음

> **카테고리 안내**:
> - 차팅 → 치료사차트·임상경과에서 사용
> - 처방 → 처방 탭에서 사용
> - 원장님 → 원장 전용 메모에서 사용
> - 일반 → 공통 사용

## 7. 진료 도구 화면 구성 요약

```
/admin/doctor-tools
├── [상용구 탭]         admin/manager 전용 — 텍스트 상용구 CRUD (진료차트/펜차트 구분)
├── [처방세트 탭]       admin/manager 전용 — 약품 조합 세트 CRUD
├── [진료세트 탭]       admin/manager 전용 — 시술 조합 세트 CRUD
├── [수가세트 탭]       admin/manager 전용 — 결제 수가항목 세트 CRUD
├── [서류 템플릿 탭]    admin/manager 전용 — 문서 양식 CRUD
├── [빠른처방 버튼 탭]  admin/manager 전용 — 빠른처방 버튼 CRUD
└── [진료 환자 목록 탭] 전체 공개 — 오늘 접수 환자 + 빠른처방 + 임시/확정 관리
```

## 자주 묻는 질문 (FAQ)

**Q. 진료차트 버튼이 안 보여요.**

A. 진료차트는 대시보드 칸반에서 고객 이름 우클릭(→ 팝업 메뉴) 또는 환자 카드 클릭 후 상세 시트의 "진료차트" 버튼으로 열립니다. 고객관리·예약관리에서도 같은 방식으로 열 수 있습니다.

---

**Q. 빠른처방 버튼 목록이 비어 있어요.**

A. admin 또는 manager 계정으로 `진료 도구` → `빠른처방 버튼` 탭에서 버튼을 등록해야 합니다.
버튼에는 처방세트를 반드시 연결해야 실제 처방이 입력됩니다.

---

**Q. 처방 확정을 실수로 눌렀어요. 취소할 수 있나요?**

A. 현재 확정 취소 기능은 없습니다. admin 계정으로 Supabase 직접 수정이 필요합니다.
운영팀(장쳰 슬랙봇)에 문의해 주세요.

---

**Q. `//` 입력했는데 자동완성 팝오버가 안 떠요.**

A. 다음을 순서대로 확인하세요:

1. 진료차트의 **"임상경과"** 입력창에서 입력하고 있는지 확인 (다른 필드에서는 작동 안 함)
2. `진료 도구` → `상용구` 탭에서 **유형이 "진료차트"로 설정된** 상용구가 등록되어 있는지 확인
3. 상용구의 **단축어(shortcut_key) 필드가 채워져 있는지** 확인

---

**Q. 임상경과에만 `//` 단축어가 되나요?**

A. 네, 현재는 진료차트의 **임상경과** 입력창에서만 `//` 트리거가 작동합니다.
다른 탭(치료사차트·처방·서류)에서는 우측 패널의 **상용구 탭**(체크박스 선택)을 이용하세요.

---

**Q. 상용구를 '원장님' 카테고리로 설정하면 어디에 사용되나요?**

A. 진료차트 Drawer의 **원장 전용 메모** 영역(진료차트 패널 내 '원장님' 섹션)에서 상용구 불러오기 시 표시됩니다. 치료사 계정에는 이 섹션이 보이지 않습니다.
