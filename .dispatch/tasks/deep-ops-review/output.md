## 구체적 개선 제안

### 1. 결제 모듈 (DK-1, DK-2)
```
완료 단계 환자 → 결제 시작 버튼 →
├─ 시술 항목 + 금액 확인
├─ 할인 적용 (정률/정액)
├─ 결제 수단: 카드 / 현금 / 분할
├─ 할부 선택 (카드 시)
└─ 결제 완료 → 영수증/메모
```

### 2. 예약 수정 (TM-1)
- 주간 캘린더에서 예약 카드 클릭 → 상세 모달 → "시간 변경", "날짜 변경", "취소" 버튼 추가

### 3. 환자 상태 이동 (CO-1)
- 카드 상세 패널에 "다음 단계로 이동" 버튼 추가 (대기→상담, 상담→시술대기 등)
- 또는 드래그 앤 드롭으로 칸반 보드 이동

### 4. 시술사 필터 (PR-1)
- 시술 컬럼 상단에 "내 시술실" 드롭다운 또는 "시술 3번만 보기" 필터

### 5. 일마감 (DK-4)
```
오늘 매출 현황
├─ 총 환자 N명 (체크인 기준)
├─ 시술 완료 N명
├─ 총 매출: X원 (카드 Y원 / 현금 Z원)
├─ 시술별 건수/매출
└─ 미결제 N명 목록
```
